# Teyler & DiScenna 1986 — Hippocampal Memory Indexing Theory

Proposed and theoretically grounded the hippocampal-indexing account: the hippocampus
"does not contain the content of an experience but it does provide an index that
allows the content to be retrieved." During encoding, neocortical areas process the
features of experience; the hippocampus records which cortical regions were co-active,
forming a sparse pointer-set. Retrieval is two-stage: a partial cue activates part
of the hippocampal index, which projects back to the relevant neocortical regions and
reinstates the full stored pattern (pattern completion). Later confirmed by engram
labeling: dentate gyrus tags ~6% of eligible cells per context — deliberately sparse.
The KPM = sparse index; `research/` = rich neocortical store is the direct
engineering analogue.
