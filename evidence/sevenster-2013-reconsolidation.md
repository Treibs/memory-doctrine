# Sevenster, Beckers & Kindt 2013 — Prediction Error Governs Pharmacological Amnesia

Sevenster & Kindt's *Science* paper provided the first *causal* demonstration that
prediction error — not retrieval per se — is the necessary gate on memory reconsolidation.
Using propranolol (a β-adrenergic blocker that disrupts noradrenergic re-stabilization of a labile trace (distinct from protein-synthesis inhibition)), the experiment showed: (1) retrieval without prediction error → memory
stays stable, propranolol has no effect; (2) retrieval with prediction error (omission)
→ memory destabilizes, propranolol erases it. The PE requirement is a causal boundary
condition, not a correlate.

Used by [[F3-surprise-principle]] as the empirical ground-truth for the doctrine's
three-zone write policy: moderate δ → reconsolidate (the PE window is active); large
sustained δ → new learning (a second memory forms, not reconsolidation of the first).
Also directly grounds [[D2-novelty-gated-write]].
