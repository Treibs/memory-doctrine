# Nader, Schafe & LeDoux 2000 — Fear memories require reconsolidation after retrieval

Establishes the existence and protein-synthesis-dependence of reconsolidation: a
retrieved, consolidated memory re-enters a labile state. This is the mechanistic
foundation for [[D2-novelty-gated-write]]'s write-on-retrieval claim.

The causally decisive design: retrieval is the only difference between the amnesic
and the intact-memory groups. Everything else (age of memory, anisomycin dose,
injection site) is held constant. Therefore retrieval is what opened the write window.

The molecular layer (protein synthesis) is an analogy limit: a KPM has no protein
synthesis. The transferable claim is: **retrieval → trace labilization → update
window → re-stabilization required**. [[sevenster-2013-reconsolidation]] adds the
gate: the update window opens only under prediction error.
