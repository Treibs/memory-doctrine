# Braak & Braak 1991 — Neuropathological staging of Alzheimer's

Establishes that the memory-index hub (entorhinal cortex) is the first and most
silently degraded structure in Alzheimer's disease. [[D3-consolidation-mtt-safe]]
uses this as the primary source for the safety clause: promote-and-detach builds
the exact fragility that Alzheimer's exploits.

The KPM implication of Braak staging:

1. **Hub-first, silent degradation** — the structure that routes all index→store
   retrieval degrades before any behavioral symptom. A KPM that relies on the hub
   for promoted/detached axioms will silently lose retrieval access without any
   early warning signal.

2. **Cascade direction** — degradation moves from the hub outward, so a compromised
   hub can progressively sever access to the rest of the graph. This makes hub
   integrity a *leading* health indicator, not a trailing one.

3. **Architecture lesson** — the hub-first failure pattern recommends redundancy at
   the hub (backed-up index) and promote-and-keep-indexed as the safe promotion
   policy: if the hub survives, promoted items benefit from the index; if the hub
   degrades, promoted items are still reachable via the doctrine tier directly.
