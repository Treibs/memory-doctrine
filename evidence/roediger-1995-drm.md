# Roediger & McDermott 1995 — DRM False Memory Paradigm

The canonical demonstration that spreading-activation / gist storage is intrinsically confabulatory: studying a set of semantic associates produces confident "Remember" judgments for the never-presented critical lure at rates statistically indistinguishable from true recall. Builds on Deese (1959) and is used by [[C3-confident-but-wrong]] to ground the source-typed retrieval requirement: any spreading-activation or GraphRAG layer will generate plausible interpolations that must be flagged as *hypotheses*, not stored facts.
