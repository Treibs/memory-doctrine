# McClelland, McNaughton & O'Reilly 1995 — Complementary Learning Systems (CLS)

Formal connectionist analysis demonstrating that catastrophic interference makes it
impossible for a single network to learn new specific episodes rapidly without
overwriting previously acquired general structure. The solution is two complementary
systems with different learning rates: a **fast hippocampal system** (high learning
rate, sparse coding, one-shot binding, pattern-separated representations) paired with
a **slow neocortical system** (low learning rate, overlapping distributed
representations, statistical regularities abstracted across many exposures). The
hippocampal system feeds the neocortical one via offline interleaved replay,
gradually consolidating specific memories into general knowledge without interference.
This is the biological proof of the index/store split's necessity: merging the two
systems into one destroys one function or the other.
