# McGaugh 2004 — The Amygdala Modulates Consolidation

Provides the causal evidence that salience (arousal → stress hormones → BLA noradrenergic activation) gates consolidation strength. Memory is not a uniform recorder: it invests its consolidation budget in proportion to significance. Used by [[C4-salience-gating]] to ground the promotion-vs-confidence distinction: salience buys durable encoding (a write-time gate), not accuracy or evidential truth.
