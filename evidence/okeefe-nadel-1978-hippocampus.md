# O'Keefe & Nadel 1978 — The Hippocampus as a Cognitive Map

The foundational monograph establishing the hippocampus as both a spatial map and the
substrate of episodic memory. O'Keefe & Nadel's *locale* system (hippocampal, allocentric,
metric) contrasts with the *taxon* system (route-based, egocentric, habitual). The locale
system provides context for episodic encoding and enables pattern completion: place cells
"provide the spatial context for a memory by recalling the neural representation of the
environment in which the memory occurred."

Used by [[F4-cognitive-map-unification]] as the primary source for the claim that the
hippocampal index (B4) and the metric geometry (B1) are the same object — the cognitive
map is simultaneously both. The limits field records the Tolman behavioral replication
caveat honestly: the doctrine's confidence in spatial grounding now rests on physiology
(50+ years of replicated place-cell recordings, Nobel 2014), not on the contested
behavioral shortcutting demonstration.
