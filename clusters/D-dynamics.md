# D · Dynamics

Retrievability decays power-law from disuse (and from retrieval-induced competition and directed forgetting), but evidential confidence does not decay from disuse — a dormant axiom is dormant, not doubted. Retrieval re-opens the memory trace, yet only prediction error (mismatch) gates an actual write; re-confirmation merely reinforces retrievability while genuine surprise triggers integration or — when large — minting a new node rather than overwriting. Consolidation is a scheduled offline replay pass that promotes durable verified nodes without detaching them from the index (the MTT-safe clause: promote-and-keep-indexed, never promote-and-detach). The revision operators govern how the justification graph shrinks or suppresses beliefs when contradictions or goal context demand it.
